# 话说APP - 完整使用手册（非程序员版）

> 🌟 **欢迎使用话说APP！** 这是一个温暖的情绪树洞应用，让您可以安全地表达情感，获得AI的温柔回应，还能和朋友们分享温暖拥抱。

**本手册专为没有编程经验的朋友编写，语言通俗易懂，步骤详细清晰！**

---

## 📱 一、话说APP是什么？

### 1.1 核心功能

话说APP就像您的**私人情绪日记本** + **AI情感助手** + **温暖分享平台**：

- 📝 **写心事**：把心里的想法写下来，不用担心被评判
- 🤖 **AI回应**：AI会生成安慰您的诗句和表情包
- 💝 **收到拥抱**：朋友可以给您的心事卡片送拥抱
- 🌈 **分享温暖**：把温暖的心事分享给关心的人

### 1.2 适合谁用？

- 想找个安全地方倾诉心事的人
- 喜欢写日记但不想太正式的人
- 想和朋友分享心情的人
- 需要AI温柔安慰的人

## 🚀 二、快速开始（3步上手）

### 第一步：打开应用

**方式1：在浏览器中打开**
1. 打开浏览器（推荐Chrome、Edge、Safari）
2. 输入网址：`https://你的域名.com`（这是部署后的网址）
3. 看到话说APP的欢迎页面

**方式2：像APP一样使用（推荐！）**
1. 在手机上打开话说APP的网页
2. 点击浏览器菜单中的"添加到主屏幕"或"安装应用"
3. 手机上会出现话说APP的图标，就像普通APP一样！

### 第二步：创建您的身份

1. 打开应用后，点击"创建新身份"
2. 输入您的昵称（2-8个字，比如"小明"、"快乐的我"）
3. 选一个喜欢的表情（😊、🎉、🌟 等）
4. 点击"创建"，完成！

> 💡 **小贴士**：
> - 您可以创建多个身份（比如"工作时的我"、"生活时的我"）
> - 每个身份的心事是完全分开的，互不影响
> - 昵称一旦创建就不能改了（这是为了保护隐私）

### 第三步：写下您的第一个心事

1. 点击"写心事"按钮
2. 在文本框里输入您的心情或想法（1-1000字）
3. 点击"发布"
4. AI会自动分析您的情绪，并生成：
   - 📜 一首安慰您的诗
   - 🎭 三种表情包文案（安慰/吃瓜/损友）

## 📖 三、详细功能说明

### 3.1 身份管理 👤

#### 什么是"身份"？
身份就像您的不同"人设"。比如：
- 身份1：工作中认真的我
- 身份2：生活中搞笑的我
- 身份3：深夜emo的我

每个身份的心事完全独立，就像不同的日记本。

#### 如何操作？

**创建新身份**
1. 在主页面点击"管理身份"
2. 点击"＋ 创建新身份"
3. 输入昵称（2-8个字）
4. 选一个表情
5. 点击"创建"

**切换身份**
1. 在身份列表中点击想用的身份
2. 应用会自动切换到该身份
3. 您只会看到这个身份的心事

**归档身份（暂时不用）**
1. 点击身份旁边的"归档"按钮
2. 该身份会被隐藏，但数据还在
3. 随时可以恢复

**删除身份（永久删除）**
1. 先归档该身份
2. 点击"删除"按钮
3. ⚠️ **注意**：删除后数据无法恢复！

> 🛡️ **保护规则**：
> - 至少保留1个活跃身份
> - 只能删除已归档的身份
> - 删除前会有确认提示

---

### 3.2 写心事 📝

#### 发布心事
1. 点击"写心事"按钮
2. 输入您的心情或想法（最少1个字，最多1000字）
3. 点击"发布"
4. AI会自动分析情绪并生成回应

#### 情绪频段
发布后，AI会给您的心事打一个"情绪分数"：
- **0分**：开心、正面情绪 😊
- **1分**：平静、中性情绪 😐
- **2分**：难过、负面情绪 😢

**您可以修改这个分数**（只能改1次）：
1. 找到想修改的心事
2. 点击情绪分数旁边的"编辑"
3. 改为0、1或2
4. 点击"保存"

#### 查看心事列表
1. 在主页面可以看到您的所有心事
2. 按时间倒序排列（最新的在最上面）
3. 可以看到每条心事的：
   - 发布时间
   - 情绪分数
   - 是否已归档

#### 归档和删除心事
**归档**（暂时隐藏）
1. 点击心事的"归档"按钮
2. 归档后不在主列表显示
3. 可以在"已归档"里查看

**删除**（永久删除）
1. 先归档该心事
2. 点击"删除"按钮
3. ⚠️ 删除后无法恢复！

---

### 3.3 AI回应 🤖

当您发布心事后，AI会自动生成：

#### 1. AI诗句 📜
- 根据您心事的情绪生成一首小诗
- 开心的事 → 欢快的诗
- 难过的事 → 安慰的诗

#### 2. 三种表情包文案 🎭
- **💝 安慰型**：温柔安慰您
- **🍉 吃瓜型**：轻松幽默调侃
- **😜 损友型**：朋友式的善意调侃

您可以选择最喜欢的一种用在卡片上！

---

### 3.4 生成分享卡片 🎴

#### 什么是分享卡片？
分享卡片就像一张精美的图片明信片，包含：
- 您的心事（可选）
- AI生成的诗句
- 选中的表情包文案
- 一个二维码（扫码可以看更多）
- 拥抱计数器

#### 如何生成卡片？
1. 找到想分享的心事
2. 点击"生成卡片"按钮
3. 选择喜欢的表情包类型（安慰/吃瓜/损友）
4. 等待生成完成
5. 预览卡片效果

#### 导出卡片
1. 点击"导出卡片"
2. 选择格式（推荐PNG图片格式）
3. 图片会下载到您的手机/电脑
4. 可以发给朋友或发到社交媒体！

#### 分享卡片
1. 点击"分享"按钮
2. 选择分享方式：
   - 📱 系统分享（微信、QQ、微博等）
   - 🔗 复制链接（复制卡片链接）
   - 📋 复制文案（复制卡片的文字内容）
   - 💾 保存图片（下载卡片图片）

---

### 3.5 拥抱功能 💝

#### 什么是拥抱？
拥抱就是给朋友的心事卡片"点赞"！每次点击拥抱，计数器就会+1。

#### 如何送拥抱？
1. 打开朋友分享的卡片链接
2. 点击"送拥抱"按钮
3. 拥抱计数器+1
4. 看到"已拥抱"的提示

> ⚠️ **重要规则**：
> - 每个设备对同一个卡片，**每天只能拥抱1次**
> - 这是为了防止刷量，保证真实性
> - 明天再来可以继续拥抱！

#### 查看拥抱数
- 在卡片上可以看到拥抱计数：💝 12
- 表示有12个人给这个卡片送过拥抱

---

### 3.6 离线使用 📴

话说APP支持**离线使用**！即使没有网络，您仍然可以：

- ✅ 写心事
- ✅ 查看已发布的心事
- ✅ 管理身份
- ✅ 浏览卡片

**数据会自动保存到您的设备上**，等恢复网络后会自动同步！

### 4.2 如果您要自己部署（给技术人员）

**方式1：Vercel（最简单）**

2. 打开 https://vercel.com

1. 推送代码到GitHub仓库
2. 在仓库设置中启用Pages
3. 选择 `main` 分支
4. 完成！

#### 后端部署（API服务）

**方式1：Heroku**
1. 在Heroku创建新应用
2. 连接GitHub仓库
3. 配置环境变量（SUPABASE_URL等）
4. 部署完成！

**方式2：Docker**
```bash
# 进入api目录
cd api

# 构建镜像
docker build -t whisper-api .

# 运行容器
docker run -p 8000:8000 --env-file .env whisper-api
```

#### 必需的环境变量

创建 `.env` 文件，填入：
```
# Supabase数据库地址
SUPABASE_URL=你的supabase地址
SUPABASE_KEY=你的supabase密钥

# OpenRouter AI服务密钥
OPENROUTER_API_KEY=你的openrouter密钥
```

## ❓ 五、常见问题解答

### Q1：我的心事数据安全吗？

**非常安全！**
- 心事保存在您的设备本地
- 只有您自己能看到
- 除非您主动分享，别人看不到
- 不同身份之间完全隔离

### Q2：可以删除所有数据吗？

**可以！**
- 删除具体心事：归档后删除
- 删除身份：归档后删除
- 清空浏览器数据：清除浏览器缓存
- ⚠️ 删除后无法恢复，请谨慎操作！

### Q3：可以在多个设备使用吗？

**可以！**
- 每个设备独立使用
- 每个设备有自己的数据
- 如果需要同步，需要登录账号（未来功能）

### Q4：为什么每天只能给同一个卡片拥抱1次？

**这是为了保护真实性！**
- 防止刷量行为
- 保证拥抱数的真实性
- 明天还可以继续拥抱

### Q5：AI生成的内容可以修改吗？

**诗句和表情包文案是自动生成的，暂时不支持修改。**
但您可以：
- 重新发布心事，生成新的内容
- 选择不同风格的表情包文案

### Q6：应用突然打不开了怎么办？

**试试这些方法**：
1. 刷新页面（F5或下拉刷新）
2. 清除浏览器缓存后重试
3. 换个浏览器试试
4. 检查网络连接
5. 联系技术支持

### Q7：如何备份我的数据？

**目前支持**：
- 导出分享卡片（图片格式）
- 复制心事内容到其他地方
- 未来会推出完整的数据导出功能！

### Q8：支持哪些浏览器？

**推荐使用**：
- ✅ Chrome（最佳体验）
- ✅ Edge
- ✅ Safari（iPhone用户）
- ✅ Firefox

**不推荐**：
- ❌ IE浏览器（太老了）
- ❌ 微信内置浏览器（部分功能可能受限）
