# 广场模块使用说明

## 1. 模块介绍

广场模块（Square Module）是话说APP的社交扩展功能，提供一个公开的心事分享空间。你可以在这里看到其他用户分享的心事，也可以将自己的心事发布到广场，获得来自陌生人的温暖共鸣。

### 核心特性

- **浏览广场**：无需登录即可浏览所有心事
- **发布到广场**：将自己的心事分享给所有人
- **情绪筛选**：按心情类型筛选帖子
- **点赞（拥抱）**：用拥抱表达关心
- **评论互动**：对心事发表温暖的评论

## 2. 使用方法

### 2.1 进入广场

在底部导航栏中点击"🌌 广场"标签，即可进入广场页面。广场无需登录即可浏览。

### 2.2 浏览心事

进入广场后，你可以看到其他用户发布的心事。每条心事包含：
- 用户头像（emoji）和昵称
- 情绪标签（🌧️悲伤 / 🍃平静 / ☀️开心）
- 心事内容
- 点赞数量和评论数量
- 发布时间

### 2.3 情绪筛选

在广场页面顶部，你可以点击筛选标签来按情绪类型查看心事：
- 🌈 **全部**：显示所有心事
- 🌧️ **悲伤**：只显示低落情绪的心事
- 🍃 **平静**：只显示平静情绪的心事
- ☀️ **开心**：只显示积极情绪的心事

### 2.4 发布到广场

有两种方式将心事发布到广场：

**方式一：从卡片页发布**
1. 在"卡片"页面生成AI卡片后
2. 点击"发布到广场"按钮
3. 心事将出现在广场中

**方式二：直接发布**
1. 在广场页面点击"写下心事"（当广场为空时）
2. 填写心事内容
3. 系统自动分析情绪并发布

### 2.5 点赞（需要身份）

1. 找到想点赞的心事
2. 点击 🤗 按钮
3. 如果尚未选择身份，会弹出提示框
4. 点击"去选身份"前往设置页选择昵称和emoji
5. 返回后再次点击即可完成点赞
6. 点赞后按钮变为 🫂，表示已点赞

> 注意：同一设备对同一心事只能点赞一次

### 2.6 评论（需要身份）

1. 找到想评论的心事
2. 点击 💬 按钮展开评论区
3. 如果尚未选择身份，会弹出提示框
4. 点击"去选身份"前往设置页选择昵称和emoji
5. 返回后在输入框中输入评论内容（最多500字）
6. 点击"发送"或按回车键提交评论

### 2.7 查看评论

1. 点击心事下方的 💬 按钮
2. 评论区会展开，显示所有评论
3. 每条评论包含用户头像、昵称和评论内容
4. 如果还没有评论，会显示"还没有评论，来说点什么吧"

## 3. 常见问题

### Q: 为什么我不能点赞/评论？
A: 点赞和评论需要先选择身份。点击底部导航栏的"⚙️ 设置"，选择一个昵称和emoji即可。

### Q: 我点赞了但没有显示？
A: 每个设备对同一心事只能点赞一次。如果已经点赞过，按钮会显示为 🫂。

### Q: 我发布的帖子别人能看到吗？
A: 是的，发布到广场的心事对所有用户可见。请不要在心事中包含个人敏感信息。

### Q: 可以删除广场上的帖子吗？
A: 目前暂不支持删除已发布的广场帖子，请在发布前确认内容。

### Q: 评论有字数限制吗？
A: 评论内容最多500个字符。

## 4. 隐私说明

- 广场上的帖子使用你选择的昵称和emoji显示，不会暴露真实身份
- 系统会检测文字中的敏感信息（手机号、邮箱、身份证号）并提醒你
- 点赞记录通过设备ID标识，不会关联到你的真实身份
